# Boosting Model Performance with XGBoost

## Objective
This project demonstrates Gradient Boosting using XGBoost and compares its performance with Random Forest.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- XGBoost
- Matplotlib
- Jupyter Notebook

## Workflow
1. Load dataset
2. Clean and preprocess data
3. Train Random Forest model
4. Train XGBoost model
5. Compare performance
6. Analyze boosting advantages

## Results
XGBoost achieved improved predictive performance compared to Random Forest.

## Key Learnings
- Ensemble Learning
- Boosting Techniques
- Sequential Error Correction
- Model Comparison
- Classification Evaluation
