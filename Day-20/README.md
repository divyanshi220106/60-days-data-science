# Why Accuracy Alone Can Mislead You

## Overview
This project evaluates Machine Learning models using multiple evaluation metrics beyond accuracy.

## Models Used
- Decision Tree
- Random Forest

## Metrics Used
- Accuracy
- Precision
- Recall
- F1-score
- ROC-AUC

## Features
- Confusion Matrix Visualization
- Metrics Comparison
- Fraud Detection Evaluation

## Key Learning
Accuracy alone can be misleading in imbalanced datasets like fraud detection.
