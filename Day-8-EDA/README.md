# Titanic EDA Project 🚢

## Objective

The goal of this project is to perform Exploratory Data Analysis (EDA) on the Titanic dataset and discover hidden patterns, trends, and insights related to passenger survival.

---

## Dataset Used

Titanic Dataset

---

## Tools & Libraries Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Jupyter Notebook / Google Colab

---

## Exploratory Data Analysis Performed

* Data Cleaning
* Missing Value Analysis
* Summary Statistics
* Survival Analysis
* Gender-based Analysis
* Passenger Class Analysis
* Age Distribution Analysis
* Fare Outlier Detection
* Correlation Heatmap

---

## Key Insights

1. Female passengers had significantly higher survival rates compared to male passengers.

2. First-class passengers had better survival chances than lower-class passengers.

3. Most passengers were young adults between the age of 20–40.

4. Fare distribution contained several outliers, showing that some passengers paid very high ticket prices.

5. Passenger class and fare showed strong relationships with survival probability.

---

## Conclusion

EDA helps in understanding datasets before building machine learning models. Through visualizations and statistical analysis, meaningful business insights and hidden patterns can be identified.
