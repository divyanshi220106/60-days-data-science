# Day 9 – Cleaning Messy Real-World Data

## Objective
The goal of this project was to clean and prepare a real-world dataset by handling missing values, duplicates, and inconsistent formats.

## Dataset Used
Titanic Dataset

## Tasks Performed
- Identified missing values
- Filled null values using median and mode
- Removed unnecessary columns
- Checked duplicate records
- Standardized inconsistent text formats
- Saved cleaned dataset

## Key Learnings
- Importance of data quality
- Different techniques for handling missing data
- Real-world preprocessing workflow
- Data consistency and validation

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

## Output
Cleaned Titanic dataset ready for analysis and Machine Learning.
